Learning to See in the Dark

an image processing pipeline and a fcnn are developed to produce brightened versions of low-light images

Eulerian Video Magnification

a simple image processing pipeline consisting of spatial decomposition stage followed by temporal magnification and reconstruction is used to magnify minor temporal variations in a video

Visual Mic

this is an interesting use case of Eularean Video magnification. The subtle changes in the pixel values over a sound sensitive object, captured by a high-speed camera, are used to construct sound waves. They took this research even further by recovering audio from every-day cameras(not high-speed)

One Pixel Attack

a differential evolution based algorithm is used to attack a DNN just by altering one pixel value in the input image

DCIGN - Inverse Graphics CNN

this model consistes of an encoder and decoder networks and is extremely useful in teaching the encoder network to encode specific charesteristics of the input image

U-Nets for Image Segmentation

a 'Medium' page on how U-Nets are used for biomedical image segmentation. U-net consists of convolution and de-convolutionn phases. The heart of this algorithm lies in the fact that we can use the learned feature map to reconstruct the image back

A wise way of object-wise segmentation(MCG)

multi-scale combinatorial grouping: Multiple smaller resolution images are first made from the input image and then segmentation is carried over them before the results are grouped for producing a final iamge segmentation

Neuromorphic cameras

these cameras behave more similar to human eyes than the traditional cameras. This link leads to an ieee page describing how similar these cameras are to our eyes

Image to Image translations
