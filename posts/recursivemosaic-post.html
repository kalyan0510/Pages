To create recursive mosaicing videos of your own, follow this post

  1. First of all download all your images from Google Photos. These are used as small tiles (images) to construct the actual mosaic image.

    • Click here to open Google Archive
    • Deselect all services except Google Photos, further you can specify some of your photo albums
    • Proceed to "Next" and download images as ZIP (this can take lot of time)
    • Download the ZIP and extract the contents into a folder.

  2. Open this repository and download the code into your machine.

    $ cd Mosaicing

  3. From your downloaded archive, generate tile images each having 100x100 resolution. These are the images used to create Mosaics.

    $ mkdir mytiles; python images2tiles.py extracted_folder_path mytiles 100

  4. Generate indexes for the created tiles. This will generate a "mytilesindexes" file, which contains precomputed information needed for the subsequent operations.

    $ python indexImages.py mytiles

    Now we are good to create mosaics images.

  5. Check out the real time mosaic image generation by running $ python generateMosaicFromCamFeed.py mytiles

    This uses your laptop camera to display a real-time mosaic video on a window.

  6. To create the recursive videos:

    • Gather a bunch of images (wide ones preferred) and save them in a new folder (./myImgs/) in the working directory.

    • run

      $ tilesz=20
      $ python mosaicvideo.py myTiles myImgs $tilesz

      tilesz is the size of each tile in mosaic. Bigger it is, the more pixelated the mosaic looks and the smaller it is, the more time it takes
